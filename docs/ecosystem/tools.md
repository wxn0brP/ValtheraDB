# Other Tools

| Tool | What it does | Repo |
|------|-------------|------|
| `@wxn0brp/db` | Meta-package - bundles core + dir adapter + client for zero-config use | [GitHub](https://github.com/wxn0brP/ValtheraDB) |
| `@wxn0brp/db-core` | Core engine - provides ValtheraClass, Collection, Relation, operators, types | [GitHub](https://github.com/wxn0brP/ValtheraDB-core) |
| `@wxn0brp/db-resolver` | Auto-selects storage adapter via config, env var (`VALTHERA_<NAME>`), or `force` option | [GitHub](https://github.com/wxn0brP/ValtheraDB-resolver) |
| `ValtheraDB-e2e` | Compliance test suite - runs predefined tests against any ActionsBase adapter | [GitHub](https://github.com/wxn0brP/ValtheraDB-e2e) |
| `ValtheraDB-benchmark` | Cross-adapter CRUD benchmarks with web visualization | [GitHub](https://github.com/wxn0brP/ValtheraDB-benchmark) |
| `ValtheraDB-squirrel` | AP routing layer - hash-based distribution over unreliable storage nodes | [GitHub](https://github.com/wxn0brP/ValtheraDB-squirrel) |
| `@wxn0brp/db-client` | Remote client (`ValtheraRemote`) - proxies operations via HTTP to a ValtheraDB server | [GitHub](https://github.com/wxn0brP/ValtheraDB-client) |
| `@wxn0brp/db-string-query` | Parses queries in SQL-like (`SELECT *`) syntax | [GitHub](https://github.com/wxn0brP/ValtheraDB-string-query) |
