# Extensions

Extensions add functionality on top of ValtheraDB - replication, indexing, and concurrency control.

| Package | What it does | Install | Repo |
|---------|-------------|---------|------|
| `@wxn0brp/db-crdt` | CRDT replication and sync between ValtheraDB instances | `bun add @wxn0brp/db-crdt` | [GitHub](https://github.com/wxn0brP/ValtheraDB-crdt) |
| `@wxn0brp/db-lock` | File-based multi-process locking with stale detection | `bun add @wxn0brp/db-lock` | [GitHub](https://github.com/wxn0brP/ValtheraDB-lock) |
